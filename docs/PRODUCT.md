## 核心定位

**小艺・全场景生活实践能力教学智能体**

聚焦AI时代不可被大模型替代的**生活实践能力、工具应用能力、场景化问题解决能力**，以自然对话为核心交互，以自适应双角色互动为教学载体。

## 核心场景

### 场景1：碎片化学习
```
用户说："教我用Excel做数据分析"
  ↓
AI听懂 → 温柔回应 → "数据分析是很实用的技能！我们从简单开始"
  ↓
拆解为3分钟可完成的小步骤 → 每步后练习 → 即时反馈
```

### 场景2：温故知新
```
用户说："我再复习一下Excel"
  ↓
AI读取历史记录 → "您上次学到数据透视表，我们从那里继续"
  ↓
针对薄弱点重点复习 → 生成类似练习
```

### 场景3：生活问题随时问
```
用户说："怎么用微信交水电费？"
  ↓
AI即时解答 → 步骤式引导 → "点击我→支付→生活缴费→选择账单类型"
  ↓
用户完成后 → "太棒了！您已学会微信生活缴费～"
```

## 核心能力矩阵

| 能力 | 说明 | 用户感知 |
|------|------|---------|
| **技能拆解** | 大技能拆为3分钟可学的小步骤 | "不用花很长时间" |
| **能力图谱** | 记住用户学过什么、什么不熟 | "它记得我的进度" |
| **自适应难度** | 根据用户水平调整教学速度 | "不会太难也不会太简单" |
| **即时反馈** | 练习后立即告诉对错 | "马上知道自己学得怎么样" |
| **温情鼓励** | 不用冷冰冰的"正确/错误" | "像老师也像朋友" |
| **主动复习** | 定期提醒复习薄弱点 | "帮助我巩固记忆" |

## 差异化壁垒

### 对比小艺/小爱

| | 小艺/小爱 | 我们的产品 |
|--|----------|------------|
| **定位** | 执行命令 | **教会你** |
| **记忆** | 单次会话 | **永久记住你的学习进度** |
| **交互** | 你问我答 | **主动教学 + 练习 + 反馈** |
| **温度** | 机械回应 | **像老师也像朋友** |

### 技术壁垒

1. **能力图谱** — 追踪用户技能掌握度，智能推荐学习路径
2. **自适应教学** — 根据用户水平动态调整难度
3. **温情鼓励系统** — WarmService复用，让学习有温度
4. **模式学习器** — 复用PatternLearner，观察学习习惯，优化教学

## 教学交互设计

### 典型对话流程

```
用户：教我用Excel做数据分析

AI：数据分析是很实用的技能！让我们从简单开始～

[第一步：认识Excel]
AI：首先，我们来认识一下Excel。它是一个"表格工具"，可以用来整理数据、做计算、画图表。

小练习：Excel是由什么组成的？
A. 只有行  B. 只有列  C. 行和列  D. 不知道

用户：C

AI：回答正确！Excel确实是由行和列组成的。每一格叫做"单元格"，我们可以在里面输入文字、数字或者公式。

[第二步：单元格操作]
AI：现在我们来试试实际操作。打开Excel，随便点击一个格子，输入"你好"然后按回车。

您成功了吗？
A. 成功了  B. 遇到问题了

用户：A

AI：太棒了！您已经迈出了第一步！

[继续步骤3...]
```

### 温情鼓励文案示例

| 场景 | 冷冰冰 | 温情鼓励 |
|------|--------|---------|
| 答对 | ✓ 正确 | 太棒了！继续保持～您学得真快！ |
| 答错 | ✗ 错误 | 没关系，我们再来一次！提示一下：{hint} |
| 完成步骤 | 进度50% | 您已经完成一半啦！休息一下还是继续？ |
| 完成技能 | 已学会 | 恭喜！您已经掌握了{技能名}！又进步了！ |
| 需要复习 | 薄弱点：xxx | {name}，之前学的有点忘了，要不要复习一下？ |
