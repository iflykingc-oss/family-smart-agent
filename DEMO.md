## 一、演示场景

### 场景1：第一次学习"Excel基础"

```
用户：教我用Excel
AI：👋 你好！我是小艺・全场景生活实践能力教学智能体！
    我可以教你学会各种生活技能：📊 Excel数据分析、💬 微信使用、📱 鸿蒙设备协同
    试试说："教我用Excel" 或 "我想学微信支付"

用户：教我用Excel做数据分析
AI：🎯 开始学习「Excel基础」！
    预计20分钟，共4个步骤。准备好了吗？我们先从第一步开始～

AI：[导师模式] 首先，让我们来认识一下Excel。它是一个表格软件，
    就像一张大型格子纸，由行（横的）和列（竖的）组成。
    每个格子叫做"单元格"，就像小区的门牌号，每个单元格都有自己唯一的地址。
    
AI：好啦，讲解完了，我们来做个小练习试试？

AI：[练习题] 选择题：Excel中的A1表示什么？
    A. 第1行A列  B. 第A列第1行  C. 第1列第A行  D. 不确定

用户：B
AI：🎉 回答正确！完全正确！A1确实表示第A列第1行。你的理解力很强！
```

### 场景2：复习提醒（遗忘曲线）

```
用户：30分钟后提醒我复习Excel
AI：好的！已设置提醒，30分钟后我会提醒你复习Excel～
    
    [30分钟后]
AI：📚 {name}，之前学的"Excel基础"该复习啦，温故知新哦～
    点击开始复习：第3步 - 简单公式
```

### 场景3：跨设备断点续学

```
[手机端]
用户：教我用微信支付
AI：好，开始学习「微信支付」...
    [学习到第2步，手机没电了]

[平板端]
AI：检测到上次学习到"微信支付"第2步，是否继续？
    [用户点击继续]
AI：上次你学到了"微信支付"的安全性设置，我们继续～
```

## 二、核心差异化演示

### 差异化1：系统级提醒（鸿蒙生态壁垒）

```
第三方App提醒：
    ❌ "提醒：下午3点开会"（容易被忽略）

小艺教学提醒：
    ✅ "老公，下午3点要开会啦，别忘了哦～"
       "记得带好资料，早点出发哦～"
```

### 差异化2：时空能力图谱

```
传统学习App：
    - 只记录"学过什么"
    - 不了解用户学习习惯

小艺教学：
    🌙 学习时间分析：您晚上8-10点学习效率最高
    📱 设备偏好：您最常用手机学习（70%）
    📈 学习速度：较快，每次进步明显
    🔄 遗忘曲线：已学习3天，需要复习巩固
```

### 差异化3：双角色互动

```
导师模式（讲解）：
    "Excel的公式用等号开头，比如=SUM(A1:A10)就是求和～
    记住这个规律，就像数学里的等号一样！"

陪练模式（练习）：
    "来，做道题试试！以下哪个是求和公式？
    A. =AVERAGE  B. =SUM  C. =COUNT"
```
