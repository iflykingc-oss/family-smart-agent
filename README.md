## ✨ 核心功能

### 1. 碎片化技能拆解（15 个技能）
> "教我沟通技巧" → 倾听的艺术 → 清晰表达 → 反馈技巧 → 非语言沟通

### 2. 能力图谱追踪（FSRS v6）
> "您上次学到数据透视表，我们从那里继续"

### 3. 场景化答疑（10 个场景类型）
> "面试时被问到薪资期望怎么回答？" → 场景识别 → 话术生成 → 模拟练习

### 4. 进度仪表盘
> 学习统计 + 技能掌握度 + 学习趋势 + 成就系统

### 5. 双角色互动（导师↔陪练）
> 讲解概念 + 即时练习 + 温情鼓励

### 6. 鸿蒙系统级提醒
> 系统级推送 + 跨设备同步 + 断点续学
