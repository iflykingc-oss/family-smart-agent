一款基于鸿蒙生态的智能教学助手，聚焦 AI 时代核心能力需求——生活实践能力、工具应用能力、场景化问题解决能力，以自然对话为核心交互，以自适应双角色互动为教学载体。

## 🎯 产品定位

**核心差异：** 小艺/小爱只能执行单次指令，无法记住用户特征、无法做跨场景协同。我们通过「能力图谱」实现真正的个性化教学，让你越学越会。

## ✨ 核心功能

### 1. 碎片化技能拆解
3 分钟一个小步骤，随时学、随时停：
> "教我用 Excel 做数据分析" → 拆解为：认识 Excel → 单元格操作 → 简单公式 → 数据透视表 → 图表制作

### 2. 能力图谱追踪
记住你学过什么、什么不熟：
> "您上次学到数据透视表，我们从那里继续" → 智能跳过已掌握内容

### 3. 自适应教学
根据你的水平调整速度和难度：
> 新手：慢速详细讲解；有基础：快速要点覆盖

### 4. 温情鼓励系统
不用冷冰冰的"正确/错误"，像朋友也像老师：
> "太棒了！继续保持～" / "没关系，我们再来一次！"

### 5. 鸿蒙全场景适配
手机、平板、智慧屏、音箱，随时随地学
